# Portfolio

This repo is dedicated to showcasing my portfolio as an undergrad studying Computer Engineering.

## Tech Stack

- **React** with **TypeScript** - Modern UI framework with type safety
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework for styling

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## GitHub Pages Deployment

This project is configured for automatic deployment to GitHub Pages via GitHub Actions. The workflow will automatically build and deploy your site when you push to the `main` branch.

To enable GitHub Pages:
1. Go to your repository settings
2. Navigate to Pages
3. Under "Source", select "GitHub Actions"

The site will be available at: `https://[your-username].github.io/portfolio/`
